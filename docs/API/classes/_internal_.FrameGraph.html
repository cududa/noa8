    whenReadyAsync

    • whenReadyAsync(timeStep?: number, maxTimeout?: number): Promise<void>
      Experimental

      Returns a promise that resolves when the frame graph is ready to be executed. In general, calling “await buildAsync()” should suffice, as this function also waits for readiness by default.

      Parameters

      • OptionaltimeStep: number

        Time step in ms between retries (default is 16)

      • OptionalmaxTimeout: number

        Maximum time in ms to wait for the graph to be ready (default is 10000)

      Returns Promise<void>

      The promise that resolves when the graph is ready
