    bindRenderTarget

    • bindRenderTarget(
          renderTarget?: FrameGraphRenderTarget,
          debugMessage?: string,
          applyImmediately?: boolean,
      ): void
      Experimental

      Binds a render target texture so that upcoming draw calls will render to it Note: it is a lazy operation, so the render target will only be bound when needed. This way, it is possible to call this method several times with different render targets without incurring the cost of binding if no draw calls are made

      Parameters

      • OptionalrenderTarget: FrameGraphRenderTarget

        The handle of the render target texture to bind (default: undefined, meaning "back buffer"). Pass an array for MRT rendering.

      • OptionaldebugMessage: string

        Optional debug message to display when the render target is bound (visible in PIX, for example)

      • OptionalapplyImmediately: boolean

        If true, the render target will be applied immediately (otherwise it will be applied at first use). Default is false (delayed application).

      Returns void
