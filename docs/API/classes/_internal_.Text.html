    Class Text

    noa.text - 3D text rendering subsystem (optional)

    Provides factory methods for creating 3D text meshes in the world. Uses meshwriter-cudu for text-to-mesh conversion with Babylon.js.

    This module is optional - it only initializes if meshwriter is available. If meshwriter is not installed, noa.text.ready will be false and text creation methods will return null.

    This module uses the following default options (from the options object passed to the Engine):

    {
        defaultFont: 'Helvetica',
        scale: 1,
        enabled: true,
    }

    createBillboardText

    • createBillboardText(content: string, options?: TextOptions): TextHandle

      Create billboarded text that always faces the camera. Useful for entity labels and floating names.

      Parameters

      • content: string

        Text string

      • Optionaloptions: TextOptions = {}

        Same as createWorldText

      Returns TextHandle

      Example

      const label = noa.text.createBillboardText('Player Name', {
          position: [10, 5, 10],
          letterHeight: 0.5,
          color: '#FFFF00',
      })

    createWorldText

    • createWorldText(content: string, options?: TextOptions): TextHandle

      Create 3D text mesh at a world position. Returns a TextHandle for manipulation and disposal.

      Parameters

      • content: string

        Text string to render

      • Optionaloptions: TextOptions = {}

        Text options

      Returns TextHandle

      • Handle for the text, or null if not ready

      Example

      const sign = noa.text.createWorldText('Welcome!', {
          position: [100, 10, 50],
          letterHeight: 2,
          color: '#8B4513',
      })
      sign.mesh.rotation.y = Math.PI / 4

    registerFont

    • registerFont(name: string, fontData: any): void

      Register a font for use with text rendering. Fonts must be registered before they can be used.

      Parameters

      • name: string

        Font name to register

      • fontData: any

        Font data (FontSpec object) from meshwriter/fonts

      Returns void

      Example

      import jura from 'meshwriter/fonts/jura'
      noa.text.registerFont('Jura', jura)

    updateText

    • updateText(handle: TextHandle, newContent: string): TextHandle

      Update text content (dispose old, create new). MeshWriter doesn't support in-place updates, so this recreates the mesh. Position, rotation, and options are preserved.

      Parameters

      • handle: TextHandle

        Existing text handle

      • newContent: string

        New text content

      Returns TextHandle

      • New handle (old is disposed)

      Example

      const newHandle = noa.text.updateText(oldHandle, 'New text content')
