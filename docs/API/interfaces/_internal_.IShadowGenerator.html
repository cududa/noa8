    Properties

    OptionaldoNotSerialize

    doNotSerialize?: boolean

    Specifies if the ShadowGenerator should be serialized, true to skip serialization. Note a ShadowGenerator will not be serialized if its light has doNotSerialize=true

    id

    id: string

    Gets or set the id of the shadow generator. It will be the one from the light if not defined

    • bindShadowLight(lightIndex: string, effect: Effect): void

      Parameters

      • lightIndex: string

        Index of the light in the enabled light list of the material owning the effect

      Returns Matrix
